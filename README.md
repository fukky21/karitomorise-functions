## Setup
```
$ make install
```

## ESLint
エラー箇所を確認する
```
$ make lint
```

## Prettier & ESLint
コードフォーマットを実行する
```
$ make lint-fix
```

## Emulators
Authentication Emulator, Functions Emulator, Firestore Emulatorを起動する
```
$ make start
```
